---
title:  "Listing installed Chocolatey packages"
date:   2018-12-17 09:48:00
categories: Software-Engineering
summary: How to check list of installed Chocolatey packages.
---

In cmd :

```bash
choco list --local-only
```

Or :

```
clist -l
```
